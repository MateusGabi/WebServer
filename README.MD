# TRABALHO DE REDES DE COMPUTADORES / FACOM-UFMS #

Desenvolvido por: (Mateus Gabi)[https://medium.com/@matt.moreira]

## Como usar ##

Compile os arquivos:

```make compile```

Execute o servidor

```~make run NUMERO_PORTA~```

```make run```

Por meio de um navegador, acesse: http://localhost:8080
